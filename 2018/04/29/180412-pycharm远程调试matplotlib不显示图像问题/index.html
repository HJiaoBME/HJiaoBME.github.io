windows用pycharm远程调试时matplotlib不显示图像问题

2018-04-29 python 0 评论

问题描述：在Windows上用pycharm远程调试时，matplotlib绘图不显示图像

windows上需要安装的软件

  • Xshell download
  • Xming download

一路默认安装就好

软件配置

1、Xshell

（1）开启X11 forwarding功能：

属性 -> 连接 -> 隧道 -> 勾选“转发X11连接到(X)” -> 选中”X DISPLAY”，后面的值默认 -> 确定

（2）查询服务器localhost
Xshell 远程连接到服务器，在命令行输入命令(这里一定要进行以上设置后，再连接服务器，不然不会显示结果)

echo $DISPLAY

2、打开Xming

如果1中的第二步没有显示的话，在1的第一步完成后就打开Xming

3、pycharm

run->edit configurations->在environment variables中添加DISPLAY=localhost:10.0（根据1（1）中的查询结果设定）

测试

（1）获得python matplotlib 的后端配置

import matplotlib
print matplotlib.get_backend()

(2)根据获得的后端在代码前面加入一下命令，我的是“TkAgg”

import matplotlib
matplotlib.use('TkAgg')

(3)测试

import matplotlib
matplotlib.use('TkAgg')
import matplotlib.pyplot as plt
import numpy as np

a = np.linspace(1,10)
b = np.logspace(1,10)

fig = plt.figure()
ax = fig.add_subplot(1, 1, 1)
ax.plot(a, b)
ax.set_xlabel("a")
ax.set_ylabel("b")
plt.show()

4、注意

pycharm 远程连接调试绘图过程中，需保证Xshell一直连接到服务器，不然会出现TclError: couldn’t connect to display “localhost:10.0”，不小心关掉了Xshell,再重新连接就好了
