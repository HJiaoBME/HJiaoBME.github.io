Deep Learning for Biomedical Time Series Analysis

2018-03-18 文献阅读 0 评论

整理了一下前段时间阅读的一系列的关于深度学习用于生物医学时间序列分析的文献。

一、时间序列

1. 概念

按照时间发生先后顺序进行排列的数据点序列（维基百科）
如：股票价格、语音、EEG、ECG……

（1）单变量

T = {t1, t2, …, t3}

（2）多变量

M = {m .1, m .2, …, m .t}

m .t = {m 1t, m2t, …, m it}

2. 任务

采集的时间序列通常被用来做分类、预测和异常检测等任务，传统经典的方法会首先通过PCA、ICA、小波变换等方法，或根据先验知识进行特征提取，再根据这些特征进行相应的任务，这类方法成本大、且需要相关领域的专业知识。

现在深度学习在时间序列上也得到了广泛的应用，深度学习通常是从原始信号中自主的学习特征。

二、经典的处理时间序列的深度学习模型

1. deepCNN/Multi-Channels DeepCNN

1.1 单通道

Kiranyaz等[1]用具有三层卷积层和两层全连接层的一维卷积神经网络，实现了患者特异性的实时心电分类。该方法将传统ECG分类方法的两大块融为一体：特征提取和分类，省去了手工提取特征的步骤，且计算速度更快，使心电实时异常监测成为可能；与之前的研究相比[2]，该方法对VEB和SVEB的分类性能有所提升，但对SVEB分类的灵敏度仍有很大的提升空间。


Pourbabaee等[3]提出了一种筛查和识别阵发性房颤的方法，该方法以原始信号作为卷积神经网络的输入，进行特征提取，然后用不同的分类器（KNN、Linear SVM、 Gaussian SVM、MLP）进行分类，与端对端CNN分类器相比，将卷积神经网络学习的特征与其他分类器相结合将显着提高患者筛查系统的性能。论文中将卷积神经网络提取的特征和手工提取的特征相比，证实了神经网络学习的特征对阵发性房颤筛查的有效性。

Pyakillya等[4]用以下一维卷积神经网络结构同时完成了ECG特征提取和分类：
7 1D-convolutional layers + max-pooling and dropout after every layer + 3 FCN layers + dropout after every layer + softmax layer with 4 outputs (normal sinus rhythm, arrhythmic, other kind of rhythm, very noisy)

吴恩达的团队[5] 收集并标注了来自29163名患者的64121份心电图数据，以200 Hz的频率采样，构建了大规模的训练集和校验集，用来训练神经网络。其构建了34层神经网络，包括33个卷积层、一个全连接层和softmax，以原始心电图时间序列为输入，每秒输出一次预测标签。算法的准确率与受过专业训练的人类心脏病专家相当，在很多情况下甚至比人类更好

1.2 多通道

Zheng等[6]提出了用多通道卷积神经网络分类时间序列方法。该方法通过不同的卷积神经网络学习不同变量时间序列的特征，然后将学习到的特征拼接在一起，馈送到全连接层和分类其进行分类。该方法在训练时间上、分类结果精度上都具有优势，且研究证明，网络结构越深，分类效果越好。

Jin等[7]也提出了类似的用于生物医学时间序列分类的集成学习方法。

Schirrmeister等[8]提出一种基于卷积神经网络的EEG解码和可视化深度学习方法。该方法的卷积神经网络有4个卷积块。第一个卷积块对44通道的脑电信号分别在时域进行一维卷积，对学习到的特征再在空间进行卷积，再池化；第2、3、4卷积层都由一卷积层和一池化层组成；最后将该深度卷积神经网络学习到的特征馈送到线性分类器进行分类。该方法从原始脑电信号中自动提取特征，只能达到和FBCSP(“Filter bank common spatial pattern)方法近似的准确度，但是 ConvNets不仅是脑电解码方法中一种新颖有前景的方法，而且还结合了创新的可视化技术，也可能为基于脑电图的脑图开辟新的窗口。

2. RNN / LSTM

循环神经网络 (Recurrent neural networks, RNN)能够反映序列前后关联信息，将上几个时刻的隐含层数据作为当前时刻的输入, 从而允许时间维度上的信息得以保留. RNN 的网络结构如下图所示. 隐含层的结果 yj 通过参数 w 作为系统输出, 同时上一时刻的 yj (t - 1) 作为输入, 输入到当前时刻的系统中。

LSTM 接受上一时刻的输出结果、当前时刻的系统状态和当前系统输入, 通过输入门、 遗忘门和输出门更新系统状态并将最终的结果进行输出，能够学习长期依赖信息 （视频行为识别，语音分类，翻译）和学习更高层次的时间特征。

Lipton等[9]提出了一种用LSTM循环神经网络进行学习诊断的方法。该方法用了12个小时到几个月不等的多变量时间序列的临床数据，包括小孩的舒张压和收缩压、外周毛细血管重建率、呼气末二氧化碳、吸入氧分数、格拉斯哥昏迷量表、血糖、心率pH值、呼吸频率、血氧饱和度、体温和尿量等，对常见的128类疾病进行诊断。

Williams等[10]将深度学习和迁移学习应用于脑电信号的分类，分为被试是在查看人脸、场景或文字数据。该文献中用到了4种深度学习模型，分别如下，并进行比较

Biswal等[11]基于RNN提出了一种睡眠自动分期系统，将睡眠分为唤醒（W）、快速眼动（R）、非REM阶段1（N1）、非REM阶段2（N2）或非REM阶段3（N3）。

3. 由SDAEs构成的深度神经网络

自动编码机 (AutoEncoder)是一种无监督学习算法, 利用反向传播算法, 让目标值等于输入值。Rahhal等[12]使用具有稀疏约束的堆叠去噪自编码器（SDAEs）以无监督的方式从原始ECG数据中学习合适的特征表示，在这个特征学习阶段之后，添加一个softmax回归层，产生所谓的深度神经网络（DNN）。该文献中提出在交互阶段，允许专家在每次迭代中标记测试集中最相关和不确定的ECG心拍，然后用于更新DNN权重。该方法进行了无监督训练学习特征和有监督的微调。所获得的结果表明，提出的方法使分类准确度有了显著提高，且训练速度块，但需要专家干预。

4. 由RBM构成的深度信念网络

限制玻尔兹曼机 (RBM)是一个关于输入(可见) 神经元 v 和输出 (隐藏) 神经元 h 之间的概率生成模型。可见层和隐藏层的神经元之间通过一个权值矩阵 w 和两个偏置向量 c 和 b 连接，在可见层神经元之间或者隐藏层神经元之间都没有连接 。深度信念网络是由RBM作为基本单元构成的深度神经网络。

Thinsungnoen等[13]提出了一种基于深度信念网络的时间序列分析方法，该方法能够用于高维和清晰的时间序列的分析，且准确度和速度都能够有所提升。

5. 迁移学习

转移学习是一种将一个数据集上训练的模型参数，作为另一个数据集训练模型的初始化参数的方法。该方法能够加快模型的收敛速度，且能够一定程度的提升深度学习模型的性能。

Isin等[14]基于迁移学习提出了一种心律失常检测的深度学习方法，将ECG分为正常、右束支传导阻滞和起搏心拍。该方法截取每个心拍的P-T间期（R波峰后200个采样点）,直接转换成256x256的图像，并将其复制3次作为RGB三个颜色通道，作为AlexNet网络的输入数据，然后用训练好的AlexNet网络的参数作为预训练参数，进行训练，测试准确率达到了92%。

6. 混合模型

混合模型是将不同的基本单元构成的神经网络进行拼接，形成的深度神经网络模型。在处理时间序列时常见的方法，是将循环神经网络于卷积神经网络进行拼接，构成的网络复杂度高，不同的基本单元能够注意到数据不同的细节特征，从而使学习到的特征更有效。

Pigou等[15]将时域卷积神经网络和LSTM混合，对视频中的手势进行识别
